**resolveBytes function** (per D-09):
- Signature: `func resolveBytes(ctx context.Context, source *embeddings.BinarySource) ([]byte, error)`

  - `SourceKindBytes`: return `source.Bytes, nil`
  - `SourceKindBase64`: decode with `base64.StdEncoding.DecodeString(source.Base64)`, wrap error with `errors.Wrap(err, "failed to decode base64 source")`

  - `default`: `errors.Errorf("unsupported source kind %q", source.Kind)`

**resolveMIME function** (per D-06):
- Signature: `func resolveMIME(source *embeddings.BinarySource) (string, error)`
- If `source.MIMEType != ""` return it directly.
- If `source.FilePath != ""`, get lowercase extension via `strings.ToLower(filepath.Ext(source.FilePath))`, look up in `extToMIME`.
- If nothing resolved: `return "", errors.New("MIME type is required: set BinarySource.MIMEType or use a file with a known extension")`

**validateMIMEModality function** (per D-07):
- Signature: `func validateMIMEModality(modality embeddings.Modality, mimeType string) error`
- Switch on modality:
  - `ModalityImage`: require `strings.HasPrefix(mimeType, "image/")`, else `errors.Errorf("image modality requires image/* MIME type, got %q", mimeType)`
  - `ModalityAudio`: require `strings.HasPrefix(mimeType, "audio/")`
  - `ModalityVideo`: require `strings.HasPrefix(mimeType, "video/")`
  - `ModalityPDF`: require `mimeType == "application/pdf"`
  - Other modalities (text): return nil (no check needed).

**convertToGenaiContent function** (per D-05, D-08):
- Signature: `func convertToGenaiContent(ctx context.Context, content embeddings.Content) (*genai.Content, error)`
- Iterate over `content.Parts`.
- For text parts: create `genai.NewPartFromText(part.Text)`.
- For non-text parts (image, audio, video, pdf):
  1. Call `resolveMIME(part.Source)` -- error if fails
  2. Call `validateMIMEModality(part.Modality, mimeType)` -- error if fails
  3. Call `resolveBytes(ctx, part.Source)` -- error if fails
  4. Create `genai.NewPartFromBytes(data, mimeType)`
- Collect all `*genai.Part` into a slice, return `genai.NewContentFromParts(parts, genai.RoleUser), nil`.
- On any error, wrap with part index context: `errors.Wrapf(err, "part[%d]", i)`.

**convertToGenaiContents function:**
- Signature: `func convertToGenaiContents(ctx context.Context, contents []embeddings.Content) ([]*genai.Content, error)`
- Iterate, call `convertToGenaiContent` for each. On error wrap with content index: `errors.Wrapf(err, "content[%d]", i)`.

**resolveTaskTypeForContent function** (per D-15, D-16):

- Step 3: return defaultTaskType (may be empty, which is fine -- API uses its default).
