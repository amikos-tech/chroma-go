`resolveMIME(source embeddings.BinarySource) (string, error)` â€” follows D-07: check `source.MIMEType` first; if empty, try extension from `source.FilePath` or URL path; fail explicitly if neither resolves.

`resolveBytes(source embeddings.BinarySource) ([]byte, error)` â€” switches on `source.Kind`:
- `SourceKindFile`: `pathutil.ValidateFilePath(source.FilePath)`, then `os.ReadFile`
- `SourceKindBase64`: `base64.StdEncoding.DecodeString(source.Data)`
- `SourceKindBytes`: return `source.RawBytes` directly
- `SourceKindURL`: should NOT be called for URL sources (return error)
- default: error

`contentToRequest(ctx context.Context, c embeddings.Content, model string, audioOpt string) (*EmbedV2Request, error)`:
- Validate exactly one part (per D-05, single modality)
- Get the single part. Based on part's Modality:
  - `ModalityText`: extract text from `part.TextData`, build `EmbedV2Request{InputType: "text", ModelName: model, Text: &TextInput{InputText: text}}`
  - `ModalityImage`, `ModalityAudio`, `ModalityVideo`: check `part.Source.Kind`:
    - If `SourceKindURL`: use `MediaSource{URL: part.Source.Data}` (per D-06)
    - Otherwise: call `resolveBytes`, base64-encode result, use `MediaSource{Base64String: encoded}` (per D-06)
    - For audio: set `AudioInput.EmbeddingOption` to audioOpt (per D-09)
  - Other modalities (e.g., PDF): return error â€” not supported by Twelve Labs

`func (e *TwelveLabsEmbeddingFunction) EmbedContent(ctx context.Context, content embeddings.Content) (embeddings.Embedding, error)`:
- Call `contentToRequest(ctx, content, resolveModel(ctx, e.apiClient.DefaultModel), e.apiClient.AudioEmbeddingOption)`
- Call `e.doPost(ctx, *req)`
- Validate response has at least 1 data item
- Convert `float64sToFloat32s(resp.Data[0].Embedding)` and return `embeddings.NewEmbeddingFromFloat32`

`func (e *TwelveLabsEmbeddingFunction) EmbedContents(ctx context.Context, contents []embeddings.Content) ([]embeddings.Embedding, error)`:
- Loop over contents (one API call per item â€” per D-05, no batching)
- Call `e.EmbedContent(ctx, c)` for each
- Collect results, return

`func (e *TwelveLabsEmbeddingFunction) Capabilities() embeddings.CapabilityMetadata`:
- Return `CapabilityMetadata` with Modalities: text, image, audio, video (per D-04)
- Intents: `IntentRetrievalQuery`, `IntentRetrievalDocument`
- `SupportsBatch: false`, `SupportsMixedPart: false` (per D-05)

`func (e *TwelveLabsEmbeddingFunction) MapIntent(intent embeddings.Intent) (string, error)`:
- Map `IntentRetrievalQuery` -> `"query"`, `IntentRetrievalDocument` -> `"document"`
- Other neutral intents: return error with "unsupported intent"
- Custom intents: pass through as string
