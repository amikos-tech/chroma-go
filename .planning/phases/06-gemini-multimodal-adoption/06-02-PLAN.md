**Test functions to implement:**

**TestCapabilitiesForModel** (GEM-01):
- Call `capabilitiesForModel("gemini-embedding-2-preview")` -- assert:
  - `len(caps.Modalities) == 5`
  - Contains all 5: `ModalityText, ModalityImage, ModalityAudio, ModalityVideo, ModalityPDF`
  - `len(caps.Intents) == 5`
  - `caps.SupportsBatch == true`
  - `caps.SupportsMixedPart == true`
  - `caps.SupportsRequestOption(embeddings.RequestOptionDimension) == true`
  - `caps.SupportsRequestOption(embeddings.RequestOptionProviderHints) == true`
- Call `capabilitiesForModel("gemini-embedding-001")` -- assert:
  - `len(caps.Modalities) == 1` and modality is `ModalityText`
  - `caps.SupportsMixedPart == false`
  - `caps.SupportsBatch == true`
- Call `capabilitiesForModel("unknown-model")` -- assert same as gemini-embedding-001 (text-only fallback)

**TestGeminiCapabilities** (GEM-01):
- Create `GeminiEmbeddingFunction` with `apiClient: &Client{DefaultModel: DefaultEmbeddingModel}` (which is now "gemini-embedding-2-preview")
- Assert `ef.Capabilities().SupportsMixedPart == true`
- Assert `len(ef.Capabilities().Modalities) == 5`
- Create with `DefaultModel: LegacyEmbeddingModel`
- Assert `ef.Capabilities().SupportsMixedPart == false`
- Assert `len(ef.Capabilities().Modalities) == 1`

**TestMapIntent** (GEM-02):
- Create `GeminiEmbeddingFunction` with default model.
- Table-driven test mapping all 5 neutral intents to expected task type strings:
  - `IntentRetrievalQuery -> "RETRIEVAL_QUERY"`
  - `IntentRetrievalDocument -> "RETRIEVAL_DOCUMENT"`
  - `IntentClassification -> "CLASSIFICATION"`
  - `IntentClustering -> "CLUSTERING"`
  - `IntentSemanticSimilarity -> "SEMANTIC_SIMILARITY"`
- For each: call `ef.MapIntent(intent)`, assert no error, assert result equals expected string.

**TestMapIntentRejectsNonNeutral** (GEM-02):
- Call `ef.MapIntent(embeddings.Intent("CODE_RETRIEVAL_QUERY"))` -- assert error
- Assert error message contains `unsupported intent`
- Assert error message contains `ProviderHints` (guides user to escape hatch)

**TestResolveMIME** (D-06):
- Explicit MIMEType set: `BinarySource{MIMEType: "image/png"}` -> returns "image/png"
- File extension fallback: `BinarySource{Kind: SourceKindFile, FilePath: "/tmp/test.jpg"}` -> returns "image/jpeg"
- PDF extension: `BinarySource{Kind: SourceKindFile, FilePath: "/tmp/doc.pdf"}` -> returns "application/pdf"
- No MIME, no file path: `BinarySource{Kind: SourceKindBytes, Bytes: []byte("data")}` -> returns error containing "MIME type is required"
- Unknown extension: `BinarySource{Kind: SourceKindFile, FilePath: "/tmp/file.xyz"}` -> returns error

**TestValidateMIMEModality** (D-07):
- Valid cases (no error): image + "image/png", audio + "audio/mpeg", video + "video/mp4", pdf + "application/pdf", text + anything (no check)
- Invalid cases (error): image + "audio/mpeg", audio + "image/png", video + "application/pdf", pdf + "image/jpeg"
- Assert error messages contain the expected pattern (e.g., "image modality requires image/*")

**TestResolveBytesKinds** (D-09):
- `SourceKindBytes`: provide `[]byte("hello")`, assert returned bytes equal.
- `SourceKindBase64`: provide base64-encoded "hello" (`aGVsbG8=`), assert decoded bytes equal `[]byte("hello")`.
- `SourceKindFile`: create a temp file with known content, provide path, assert bytes match. Clean up with `t.Cleanup`.
- `SourceKindURL`: skip with `t.Skip("requires HTTP server")` -- URL resolution is tested in integration tests.

**TestConvertToGenaiContentText**:
- Create `embeddings.Content{Parts: []embeddings.Part{{Modality: embeddings.ModalityText, Text: "hello world"}}}`.
- Call `convertToGenaiContent(context.Background(), content)`.
- Assert no error, result is not nil.
- Assert `len(result.Parts) == 1`.
- Assert `result.Role == "user"`.

**TestConvertToGenaiContentBinary**:
- Create a content with an image part: `Modality: ModalityImage, Source: &BinarySource{Kind: SourceKindBytes, Bytes: []byte("fake-png"), MIMEType: "image/png"}`.
- Call `convertToGenaiContent(context.Background(), content)`.
- Assert no error, result has 1 part.

**TestConvertToGenaiContentMixedParts**:
- Create content with 2 parts: text + image (SourceKindBytes with MIMEType "image/jpeg").
- Call `convertToGenaiContent`.
- Assert no error, result has 2 parts.

**TestConvertToGenaiContentMissingMIME**:
- Create image part with `BinarySource{Kind: SourceKindBytes, Bytes: []byte("data")}` (no MIMEType, no FilePath).
- Call `convertToGenaiContent`.
- Assert error containing "MIME type is required".

**TestConvertToGenaiContentMIMEModalityMismatch**:
- Create image part with `BinarySource{Kind: SourceKindBytes, Bytes: []byte("data"), MIMEType: "audio/mpeg"}`.
- Call `convertToGenaiContent`.
- Assert error containing "image modality requires image/*".

**TestResolveTaskTypeForContent** (D-15, D-16):
- Create a minimal `GeminiEmbeddingFunction` with default model for the mapper.
- No hints, no intent: assert returns default task type.
- With intent `IntentRetrievalQuery`, no hints: assert returns `TaskTypeRetrievalQuery`.
- With `ProviderHints["task_type"] = "CODE_RETRIEVAL_QUERY"`: assert returns `TaskTypeCodeRetrievalQuery` (override).
- With both hint and intent: assert hint wins (D-16 order).
- With invalid hint `ProviderHints["task_type"] = "INVALID"`: assert error.

**TestEmbedContentLegacyModelRejectsMultimodal** (D-03, D-04):
- Create `GeminiEmbeddingFunction` with `apiClient: &Client{DefaultModel: LegacyEmbeddingModel}` and set `t.Setenv("GEMINI_API_KEY", "test-key")`.
- Create content with image part: `Content{Parts: []Part{{Modality: ModalityImage, Source: &BinarySource{Kind: SourceKindBytes, Bytes: []byte("fake"), MIMEType: "image/png"}}}}`.
- Call `ef.EmbedContent(ctx, content)`.
- Assert error (from `ValidateContentSupport` rejecting image modality for text-only caps).
- Assert error contains "unsupported" or is a `*embeddings.ValidationError`.

**TestGeminiContentRegistration** (GEM-03):

- Assert no error, result is not nil.
- Assert result implements `embeddings.ContentEmbeddingFunction` (already proven by return type).
- Type-assert to `embeddings.CapabilityAware`, call `Capabilities()`, assert 5 modalities.
